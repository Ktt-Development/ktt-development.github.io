Complicated tasks made easy

Simplified exchange methods for:

  • Parsing HTTP POST with multipart/form-data support.
  • Sending gzip compressed responses.
  • Simplified output stream writing with #send.
{% highlight java %} SimpleHttpHandler handler = new SimpleHttpHandler(){ @Override public void handle(SimpleHttpExchange exchange){ Map POST = exchange.getPostMap(); exchange.send("OK", HttpCode.HTTP_OK, true); } }; {% endhighlight %}

Extended Features

Out of the box support for:

  • Http Cookies
  • Http Sessions
  • Multithreaded Servers
{% highlight java %} SimpleHttpServer server = new SimpleHttpServer(8080); server.setHttpSessionHandler(new HttpSessionHandler()); HttpHandler handler = new HttpHandler(){ @Override public void handle(HttpExchange exchange){ HttpSession session = server.getHttpSession(exchange); String session_id = session.getSessionID(); exchange.close(); } }; {% endhighlight %}

Simplified Handlers

Easy to use handlers:

  • Redirect Handler
  • Predicate Handler
  • File Handler
  • Server-Sent-Events Handler
  • Temporary Handler
  • Throttled Handler
{% highlight java %} RedirectHandler redirect = new RedirectHandler("https://github.com/"); FileHandler fileHandler = new FileHandler(); fileHandler.addFile(new File("index.html")); fileHandler.addDirectory(new File("/site")) SSEHandler SSE = new SSEHandler(); SSE.push("Server sent events!"); ThrottledHandler throttled = new ThrottledHandler(new HttpHandler(), new ServerExchangeThrottler()) {% endhighlight %}
